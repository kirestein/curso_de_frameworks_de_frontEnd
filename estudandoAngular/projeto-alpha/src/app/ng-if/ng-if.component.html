Implementação da Diretiva *ngIf

Propriedade "exiba":

Uso da diretiva *ngIf

Texto exibido quando o checkbox estiver "ATIVADO" (ou seja, quando for true)

Texto exibido quando o checkbox estiver "DESATIVADO" (ou seja, quando for false)

Usando *ngIf/Else

Texto exibido quando o checkbox estiver "ATIVADO" (ou seja, quando for true)

Texto exibido quando o checkbox estiver "DESATIVADO" (ou seja, quando for false)

Usando *ngIf - then - else

Texto exibido quando o checkbox estiver "ATIVADO" (ou seja, quando for true) Texto exibido quando o checkbox estiver "DESATIVADO" (ou seja, quando for false)
